## 📌 프로그램
### 💻 ``ArrowScroll_CHAR``
  C 스타일로 구현하여 문자열을 char형 배열에 받고 문자열을 저장하는 컨테이너로 포인터배열을 사용

### 💻 ``ArrowScroll_STRING``
  문자열을 string으로 받고 문자열을 저장하는 컨테이너로 STL의 list를 활용하여 구현
